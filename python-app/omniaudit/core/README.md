# Core Module

Core functionality for OmniAudit including plugin management, configuration loading, and event bus.

## Components (Planned)

- **plugin_manager.py** - Dynamic plugin discovery and lifecycle management
- **config_loader.py** - Configuration file parsing and validation
- **event_bus.py** - Event-driven communication between components

## Phase

Phase 1 (Current): Stubs
Phase 2: Full implementation with plugin discovery
